# fastify-boilerplate

## Available Scripts

In the project directory, you can run:

### `npm run start`

For production mode

### `npm run start:dev`

For development mode

### `npm run format`

For formatting code

### `npm run lint`

For running code linter

To start the app in dev mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
3000 is default port
